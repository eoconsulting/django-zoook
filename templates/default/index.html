Django configuration

  1. 1. In static dir, create symbolic link at inplaceeditform library: static$ ln -s /usr/local/lib/python2.6/dist-packages/django_inplaceedit-0.69-py2.6.egg/inplaceeditform/media/ inplaceeditform
  2. 2. Add in config.py your configuration (database, OpenERP connection, sale, ....)
  3. 3. py files in sync dir: change execute permisions
  4. 4. Edit config_path.py. Add you Django path
  5. 5. Add same languages OpenERP in settings.py (LANGUAGES)
  6. 6. Create tables: python manage.py syncdb
  7. 7. Run Server: python manage.py runserver

OpenERP configuration

  1. 1. Sale/Administration/Sales. Configure your Sale Shop and paramiko connection (ssh)
  2. 2. Add Countries to Check Vat. This option is requiered because is same countries avalaible shop.
  3. 3. Sale Order or Invoices sequence not use / (for example, SO/001). URL navigator get order number and / it's a path url and broken.
